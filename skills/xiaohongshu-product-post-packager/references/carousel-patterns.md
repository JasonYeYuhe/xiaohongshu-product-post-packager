# Carousel Patterns

## Default tutorial arc

1. **Hook** — what changed and why the reader should care
2. **Value** — what this enables in daily use
3. **Capability map** — 3–5 concrete things it can do
4. **Step 1** — where to start
5. **Step 2** — the only non-obvious hurdle
6. **Step 3** — what success looks like
7. **Caveats** — platform limits, account requirements, rollout notes
8. **Close** — save / try / compare / comment prompt

## Strong hook shapes

- “终于，X 可以 Y 了”
- “这个功能一开，X 的使用方式变了”
- “我把 X 的新功能跑通了，最值得记住的是这 4 步”
- “如果你经常在路上盯长任务，这个更新很有用”

## Caption pattern

1. One-sentence hook
2. One short paragraph on why it matters
3. Numbered steps
4. Caveats
5. One-sentence close
6. Hashtags

## Editing rules

- One idea per card
- Prefer concrete verbs over abstract praise
- Keep caveats factual, not apologetic
- If a sentence only exists to sound excited, cut it
- If a sentence saves the reader a mistake, keep it
